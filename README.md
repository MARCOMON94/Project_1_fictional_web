# Circle Agency - Mid Term Project

This project is a responsive front-end website built as part of the Ironhack Mid Term Project.

It recreates the **Circle Agency** design using **HTML**, **CSS**, and **JavaScript**, including dynamic content loaded from an external API and responsive layouts for different screen sizes.

## Live Demo
[View the project online](https://fictionalwebcircleagency.netlify.app/)

## Features

- Responsive homepage
- Fixed navigation bar with hamburger menu on small screens
- Dynamic "Recent Projects" section loaded from API
- Project details page with dynamic content
- Contact form with custom validation
- Newsletter form with email redirect to contact page
- Hover effects and responsive cards
- Back to top floating button

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Netlify for deployment
